# ⚡ X Intent Generator

A **premium client-side tool** that converts any **X (Twitter) post URL** into actionable **intent links** such as:

- ❤️ Like
- 🔁 Repost / Retweet
- 💬 Reply
- 📢 Quote Tweet
- 👤 Follow Author
- 🔗 Share Tweet

Built with **Vanilla JavaScript + TailwindCSS** and designed with a **modern dark UI**.

---

# ✨ Features

✔ Convert any X post link instantly  
✔ Supports multiple URL formats  
✔ Generate all intent links in one click  
✔ Copy intent URLs to clipboard  
✔ Open intent directly in X  
✔ Automatic Tweet ID extraction  
✔ Username detection for Follow intent  
✔ Manual username fallback  
✔ Responsive modern UI  
✔ Fully client-side (No API needed)
